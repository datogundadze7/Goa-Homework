პროგრამირების ენები

Python

Python არის მაღალი დონის პროგრამირების ენა, რომელიც გამოიყენება ვებ-დეველოპმენტში, მეცნიერებაში, მონაცემების ანალიზში და ხელოვნური ინტელექტის სფეროებში. მისი სინტაქსი მარტივია და კითხვადი, რაც მას პოპულარულს ხდის დამწყებთათვის.

შეიქმნა: 1991 წელს გიდო ვან როსუმის მიერ.

მაგალითი: 

print("Hello, World!")

JavaScript

JavaScript გამოიყენება ვებ-გვერდების დინამიკური შინაარსის შესაქმნელად. ის მუშაობს როგორც კლინეტ-საიდს, ასევე სერვერ-საიდს (Node.js-ის მეშვეობით).

შეიქმნა: 1995 წელს ბრენდან აიკის მიერ.

მაგალითი: 

document.write("Hello, World!");

Java

Java არის ობიექტზე ორიენტირებული პროგრამირების ენა, რომელიც გამოიყენება ვებ-აპლიკაციების, მობილური აპლიკაციების და დიდი სისტემების შესაქმნელად.

შეიქმნა: 1995 წელს ჯეიმს გოსლინგის და მისი გუნდის მიერ.

მაგალითი: 

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

C++

C++ არის მაღალი დონის პროგრამირების ენა, რომელიც გამოიყენება სისტემური პროგრამირების, გრაფიკული აპლიკაციების და თამაშების შესაქმნელად.

შეიქმნა: 1983 წელს ბიარნე სტრუსტრუპის მიერ.

მაგალითი: 

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Ruby

Ruby არის დინამიკური, ობიექტზე ორიენტირებული პროგრამირების ენა, რომელიც გამოიყენება ვებ-დეველოპმენტში (Ruby on Rails-ის მეშვეობით) და სკრიპტების წერაში.

შეიქმნა: 1995 წელს იუკიჰირო მაცუმოტოს მიერ.

მაგალითი: 

puts "Hello, World!"
